# Final Project Storyboard

## Central Story

Formula 1 did not simply add more races over time. It grew from a mostly European racing calendar into a global sport, and that changing calendar provides the context for understanding which constructors and drivers defined each era.

## Story Structure

Use a Martini Glass structure:

1. Start with an author-guided story.
2. Walk the viewer through the key scenes in a fixed order.
3. End with an open exploration stage where the viewer can select years, constructors, and drivers.

## Scene Sequence

| Scene | Visual Focus | Main Question | Transition Purpose |
| --- | --- | --- | --- |
| 1. The question | Title and early map | What changed in F1 history? | Establish the main narrative. |
| 2. European origins | 1950s circuit map | Where did F1 begin? | Show the baseline before expansion. |
| 3. Global expansion | Animated circuit map | How did the race calendar spread? | Reveal geographic growth over time. |
| 4. Calendar becomes global | Continent timeline | When did the calendar become less Europe-centered? | Aggregate map points into a clearer time summary. |
| 5. Constructor eras | Constructor streamgraph | Which teams dominated the changing calendar? | Reuse the same time axis to connect expansion and competition. |
| 6. Driver drilldown | Driver timeline or ranking | Which drivers explain each team era? | Drill from constructor dominance into human-level detail. |
| 7. Open exploration | Linked views | What can the viewer discover? | Move from guided story into interactive exploration. |

## What To Avoid

- Do not add unrelated charts just because the dataset supports them.
- Do not jump from geography to dominance without a transition.
- Do not make the story feel like a dashboard dump.
- Do not animate everything at once. Use simple filtering and highlighting transitions.

## Instructor Feedback Fix

The storyboard should explicitly explain how each scene leads into the next:

- Map expansion leads to continent timeline because both are about the race calendar.
- Continent timeline leads to constructor dominance because both use the same seasons.
- Constructor dominance leads to driver drilldown because drivers explain who powered those teams.
- Open exploration comes last because the guided story teaches the viewer how to use the views.
